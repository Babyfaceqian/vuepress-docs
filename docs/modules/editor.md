## ckeditor
一款富文本编辑器，拥有正常、气泡和文档模式，可根据自定义配置打包以减小代码体积。支持以下功能：

- 自动格式化（markdown语法）
- 自动文本转换
- 基本文本格式（bold|italic|underline|strikethrough|code|subscript|superscript）
- 缩进
- 工具栏（固定|气泡）
- 代码块
- 文档标题
- 占位符
- 字体
- 高亮
- 水平线
- 图片
- 键盘支持
- 链接
- markdown输出
- 数学和化学公式
- 嵌入多媒体
- @提及
- 分页
- 带格式粘贴（从work|google doc）
- 只读模式
- 去除格式
- 限制输入模式
- 多语言
- 语法拼写检查
- 表格
- 文本对齐
- 待办项
- 字数统计
- watchdog（出错后内容恢复）
